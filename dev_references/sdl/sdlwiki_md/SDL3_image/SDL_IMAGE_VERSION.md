# SDL_IMAGE_VERSION

This is the version number macro for the current SDL_image version.

## Header File

Defined in
[\<SDL3_image/SDL_image.h\>](https://github.com/libsdl-org/SDL_image/blob/main/include/SDL3_image/SDL_image.h)
