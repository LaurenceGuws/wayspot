# SDL_PLATFORM_APPLE

A preprocessor macro that is only defined if compiling for Apple
platforms.

## Header File

Defined in
[\<SDL3/SDL_platform_defines.h\>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_platform_defines.h)

## Syntax

<div id="cb1" class="sourceCode">

``` sourceCode
#define SDL_PLATFORM_APPLE 1
```

</div>

## Remarks

iOS, macOS, etc will additionally define a more specific platform macro.

## Version

This macro is available since SDL 3.2.0.
