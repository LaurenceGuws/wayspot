# SDL_PropertiesID

An ID that represents a properties set.

## Header File

Defined in
[\<SDL3/SDL_properties.h\>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_properties.h)

## Syntax

<div id="cb1" class="sourceCode">

``` sourceCode
typedef Uint32 SDL_PropertiesID;
```

</div>

## Version

This datatype is available since SDL 3.2.0.

## Using Properties

While this looks like an integer to the application, SDL properties are
actually key/value stores that can manage sets of information with
multiple datatypes.

Please refer to [CategoryProperties](CategoryProperties.html) for all
the details.
