# SDL_WINDOWPOS_CENTERED_MASK

A magic value used with
[SDL_WINDOWPOS_CENTERED](SDL_WINDOWPOS_CENTERED.html).

## Header File

Defined in
[\<SDL3/SDL_video.h\>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_video.h)

## Syntax

<div id="cb1" class="sourceCode">

``` sourceCode
#define SDL_WINDOWPOS_CENTERED_MASK    0x2FFF0000u
```

</div>

## Remarks

Generally this macro isn't used directly, but rather through
[SDL_WINDOWPOS_CENTERED](SDL_WINDOWPOS_CENTERED.html) or
[SDL_WINDOWPOS_CENTERED_DISPLAY](SDL_WINDOWPOS_CENTERED_DISPLAY.html).

## Version

This macro is available since SDL 3.2.0.
