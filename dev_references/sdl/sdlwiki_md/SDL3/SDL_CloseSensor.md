# SDL_CloseSensor

Close a sensor previously opened with
[SDL_OpenSensor](SDL_OpenSensor.html)().

## Header File

Defined in
[\<SDL3/SDL_sensor.h\>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_sensor.h)

## Syntax

<div id="cb1" class="sourceCode">

``` sourceCode
void SDL_CloseSensor(SDL_Sensor *sensor);
```

</div>

## Function Parameters

|  |  |  |
|----|----|----|
| [SDL_Sensor](SDL_Sensor.html) \* | **sensor** | the [SDL_Sensor](SDL_Sensor.html) object to close. |

## Version

This function is available since SDL 3.2.0.
