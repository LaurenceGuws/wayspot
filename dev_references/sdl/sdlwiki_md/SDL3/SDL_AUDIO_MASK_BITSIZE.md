# SDL_AUDIO_MASK_BITSIZE

Mask of bits in an [SDL_AudioFormat](SDL_AudioFormat.html) that contains
the format bit size.

## Header File

Defined in
[\<SDL3/SDL_audio.h\>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_audio.h)

## Syntax

<div id="cb1" class="sourceCode">

``` sourceCode
#define SDL_AUDIO_MASK_BITSIZE       (0xFFu)
```

</div>

## Remarks

Generally one should use [SDL_AUDIO_BITSIZE](SDL_AUDIO_BITSIZE.html)
instead of this macro directly.

## Version

This macro is available since SDL 3.2.0.
